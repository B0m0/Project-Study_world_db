# Study_world_db

**Study_world_db** is a MySQL-based project focused on exploring and understanding global dynamics through data on language, population, and currency. The project emphasizes data cleaning, analysis, and visualization to provide actionable insights for stakeholders.

---

## Project Overview

The project aims to reveal how language diversity, population metrics, and economic factors influence global trends. It involves:

- **Data Cleaning & Preparation:**  
  Rigorous cleaning to handle missing values, correct errors, and create a robust dataset ready for analysis.

- **Key Performance Indicators (KPIs):**  
  Calculated metrics to uncover insights on how population and language factors impact regional and global dynamics.

- **Exploratory Data Analysis (EDA):**  
  Identified patterns, relationships, and trends within the data to guide informed decision-making.

- **Interactive Dashboards:**  
  Designed user-friendly dashboards in Power BI to visualize KPIs and insights, making complex data accessible for stakeholders.

---

## Tools & Technologies

- **Database:** MySQL  
- **Data Analysis:** SQL, data cleaning, KPIs  
- **Visualization:** Power BI  
- **Data Management:** Handling missing values, correcting errors, and preparing datasets for analysis  

---

## Key Insights

- Language diversity and population metrics significantly influence global trends.  
- Dashboards provide an interactive view of regional and global dynamics.  
- The project demonstrates how structured data analysis can support informed decision-making on worldwide factors.
